# Asana Workflow for Alfred 4

You can add task to asana (`a`).

## Setup

### access token

Get your asana personal access token.

@see https://asana.com/guide/help/api/api

### Env

Set Workflow Environment Variables.

- `ASANA_PERSONAL_ACCESS_TOKEN`
- `ASANA_WORKSPACE_ID`
- `ASANA_ASSIGNEE`
  - assignee email
